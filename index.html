Sistema de información y Registro de Asistencia por Huella Digital

SIRAHD

Mas información

Introducción

Es un Sistema de información que permite el registro de los profesores, y el control de asistencias de los profesores del Departamento Físico-Matemáticas de la UASLP mediante huella digital con la opción de generar reportes de consultas específicas. Para su desarrollo, se utilizará como lenguaje de programación C#, como base de datos SQL Server Express y para su comunicación el SDK del dispositivo fingerprint reader U.are.U 4500.

Qué más ofrece

Descripción

SIRAHD es un sistema de información, que permite el registro de la plantilla de los profesores a la base de datos realizada en SQL Server Express 2017, el control de las asistencias de los profesores mediante la huella digital con parámetros variables de asistencia y retardo, visualizaciones de gráficas para el análisis de consultas específicas; tales como el número de inasistencias por día, reportes por rango de fecha en PDF y en Excel, con el uso de cuentas administrador y usuario para su uso. Se realiza una actualización en el administrador de base de datos a una versión actual, en la nueva base de datos los campos y tablas para el uso de usuarios, se agregara las huellas digitales de los profesores y el manejo de las inasistencias, y con el sistema de información se desarrollara formatos para el llenado de la base de datos y con el nuevo dispositivo lector de huella, se desarrollara e implementara el control de asistencias, también la opción de crear reportes y se generen en formato .xls y .pdf. También el uso de gráficas para la visualización de la asistencia diaria.
